# Manabu 学ぶ

According to the [Jisho online][1] dictionary, [Manabu (学ぶ)][2] means:

> ###### まな
> ### 学ぶ
> 1. to study (in depth); to learn; to take lessons in
> 
> Other forms
> 
> 學ぶ 【まなぶ】
> 
> Notes
> 
> 學ぶ: Out-dated kanji.
